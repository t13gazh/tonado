# Erste Schritte

Die Box ist zusammengebaut und steht vor dir. Was jetzt? Keine Sorge — du brauchst heute Abend noch keine zehn Dinge lernen. Die ersten fünf Schritte reichen, damit dein Kind morgen früh eine Figur auflegen und Musik hören kann.

## 1. Box einschalten

Stecker rein. Die Box braucht beim allerersten Start eine knappe Minute, bis sie vollständig bereit ist — sie muss einmal alles für sich einrichten. Bei späteren Starts geht es deutlich schneller.

Du hörst keinen Startton und siehst nur eine kleine LED leuchten. Das ist normal. Während die Box hochfährt, geh weiter zu Schritt 2.

## 2. Mit deinem Smartphone verbinden

Beim allerersten Mal kennt die Box dein WLAN noch nicht. Darum spannt sie selbst ein eigenes, kleines WLAN auf, in das du dich kurz einklinkst, um ihr dein Heim-WLAN mitzuteilen.

1. Am Handy die WLAN-Einstellungen öffnen.
2. Nach einem Netz namens **Tonado-Setup** (oder ähnlich) suchen und verbinden.
3. Dein Handy öffnet automatisch eine kleine Einrichtungs-Seite (falls nicht: Browser auf und `http://tonado.local` aufrufen).
4. Dein Heim-WLAN aus der Liste wählen, das Passwort eingeben.
5. Eine Eltern-PIN vergeben — vier bis sechs Ziffern, die du dir merken kannst. Damit schützt du später die Einstellungen.

Die Box verbindet sich jetzt mit deinem WLAN. Dein Handy springt dabei von selbst zurück ins Heim-WLAN. Nach etwa einer halben Minute ist die Box erreichbar — du landest automatisch auf der App.

## 3. Erste Musik hochladen

Damit dein Kind etwas hört, braucht die Box Musik. Am einfachsten: ein paar Lieblings-Lieder vom Handy oder Computer.

1. In der App die **Bibliothek** öffnen.
2. Auf das Plus-Symbol tippen, **Dateien hochladen** wählen.
3. Eine oder mehrere Musik-Dateien aussuchen.
4. Warten, bis der Upload fertig ist (dauert je nach Größe ein paar Sekunden).

Für den Anfang reicht ein Ordner mit drei, vier Liedern — später kannst du ganze Hörbücher oder Alben nachladen.

## 4. Erste Figur zuordnen

Jetzt kommt der schöne Moment. Eine Figur oder Karte suchen, die dein Kind mit der Musik verbinden soll.

1. In der App auf **Figuren** gehen, dann **Figur hinzufügen** tippen.
2. Einen Namen vergeben („Hörbuch Maus", „Einschlaf-Lied", was immer passt).
3. Die Figur oder Karte auf die Box legen, wenn die App dich dazu auffordert.
4. Aus der Bibliothek auswählen, was die Figur abspielen soll.
5. Figur wieder wegnehmen, wenn die App „fertig" sagt.

Das war's. Die Figur „kennt" jetzt ihren Inhalt.

## 5. Dein Kind spielen lassen

Figur auf die Box legen → Musik startet.

Nimmt dein Kind die Figur weg, pausiert die Musik (oder spielt weiter — je nachdem, wie du es in den Einstellungen wählst).

Legt dein Kind eine andere Figur auf, wechselt die Box sofort zum neuen Inhalt.

Das ist die ganze Bedienung. Dein Kind kann ab hier selbst.

## Wenn etwas nicht klappt

**Das Einrichtungs-WLAN „Tonado-Setup" taucht nicht auf.**
Warte eine Minute und aktualisiere die WLAN-Liste auf dem Handy. Falls es nach zwei Minuten noch fehlt: Stecker kurz ziehen, wieder anschließen, erneut versuchen.

**Ich habe die PIN vergessen.**
Die PIN lässt sich über den Einrichtungs-Assistenten neu setzen. Dafür brauchst du einmal direkten Zugriff auf die Box (siehe Bastler-Anleitung) — die Kurzfassung: Box im Einrichtungs-Modus starten, PIN neu vergeben.

**Die Figur wird nicht erkannt.**
Die Figur muss direkt auf der markierten Stelle liegen (nicht seitlich daneben). Falls es weiter nicht klappt: andere Figur oder Karte ausprobieren — manche alten Chips sind nicht kompatibel. Details in der [Hardware-Anleitung](../fuer-bastler/hardware.md).

**Die Musik ist zu leise.**
In den Einstellungen die Lautstärke-Obergrenze anschauen — wenn sie niedrig steht, kommt auch bei maximalem Tippen nichts Lauteres raus. Bewusste Schutz-Einstellung für Kinderohren, kein Fehler.

**Beim Update oder Neustart hakt etwas.**
Box einmal komplett vom Strom trennen, 10 Sekunden warten, wieder einstecken. Hilft das nicht, ein Issue auf [GitHub](https://github.com/t13gazh/tonado/issues) öffnen — wir schauen uns das an.
