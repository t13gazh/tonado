# Screenshots

In diesem Ordner liegen die Screenshots für die README und Doku.

## Konventionen

- **Format:** PNG bevorzugt (lossless), JPG nur für Hero-Fotos mit Hardware
- **Breite:** 1080px (Mobile-Shots) / 1920px (Desktop-Shots), unkomprimiert
- **Namensschema:** `NN-kurzname.png` — numerische Reihenfolge, kebab-case

## Aktuelle Screenshots

Wird ergänzt, sobald die ersten Shots da sind.

## Tools für Screenshots

- **Chrome DevTools** (F12 → Toggle Device Toolbar) — konsistenter Mobile-Viewport (iPhone 14, 390×844)
- **Windows Snipping Tool** / **ShareX** für Desktop-Shots
- **[shots.so](https://shots.so)** — Device-Frames (kostenfrei, kein Account)
- **[screely.com](https://screely.com)** — Browser-Frame mit macOS-Look
